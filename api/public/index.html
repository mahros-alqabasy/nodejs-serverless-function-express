Task Tracker API Documentation

GET /api/tasks

Retrieve a list of all tasks.

Request:
GET /api/tasks
            
Response:
Status: 200 OK
[
    {
        "id": 1,
        "description": "Buy groceries",
        "status": "todo",
        "createdAt": "",
        "updatedAt": ""
    },
    {
        "id": 2,
        "description": "Clean the house",
        "status": "in-progress",
        "createdAt": "",
        "updatedAt": ""
    }
]

POST /api/create

Create a new task.

Request:
POST /api/create
{
    "description": "New Task"
}
            
Response:
Status: 201 Created
{
    "message": "Task created successfully"
}

PUT /api/update/:id

Update an existing task.

Request:
PUT /api/update/1
{
    "description": "Updated Task",
    "status": "done"
}
            
Response:
Status: 200 OK
{
    "message": "Task updated successfully"
}

DELETE /api/delete/:id

Delete a task.

Request:
DELETE /api/delete/1
            
Response:
Status: 200 OK
{
    "message": "Task deleted successfully"
}

GET /api/tasks/:id

Retrieve a specific task by its ID.

Request:
GET /api/tasks/1
            
Response:
Status: 200 OK
{
    "id": 1,
    "description": "Buy groceries",
    "status": "todo",
    "createdAt": "",
    "updatedAt": ""
}
